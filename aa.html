Paid Internet Access via Local Area Network(A Secure and Controlled Network System for Internet Access.)

This project implements a controlled Local Area Network (LAN) that provides internet access to users based on a payment system. Users are required to authenticate before gaining access to the internet, ensuring a secure and managed network environment. The system includes network access control, user authentication, and a payment gateway to enable seamless prepaid or pay-as-you-go internet access. Designed for businesses, cafes, co-working spaces, and community networks, this solution ensures fair usage and revenue generation while maintaining network security.
